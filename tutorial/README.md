## 🚀 快速开始

1. 从 **01_项目初始化与目录结构.md** 开始
2. 按顺序学习每个文档
3. 每完成一个阶段，运行测试验证

## 📋 前置条件

- Go 1.20+
- MySQL 8.0+
- Redis 6.0+
- Kafka（可选，阶段六）

## 🎯 学习目标

完成本系列教程后，你将：
- 掌握 Go Web 开发的最佳实践
- 理解即时通讯系统架构
- 能够独立开发分布式聊天应用
